## LPHO BMD : Loei Provincial Public Health Office Budget Management Dashboard

## ระบบบริหารจัดการงบประมาณสำนักงานสาธารณสุขจังหวัดเลย

📊 สรุป Feature ปัจจุบันของระบบ BMD LOEI

### ✨ 1. ระบบภาพรวมและแดชบอร์ด (Executive Dashboard & Reports)

- Real-time Dashboard: แสดงยอดรวมงบประมาณ (อนุมัติ, จัดสรร, เบิกจ่าย, คงเหลือ) แยกตามกลุ่ม สป.สธ. และ เงินนอก สป. (เฉพาะเงินบำรุง สสจ.เลย)
- Visualizations: มีกราฟ Gauge Chart แสดงร้อยละภาพรวม และ Bar Chart แสดงผลงานเบิกจ่าย Top 10 ของกลุ่มงาน
- Executive Summary: รายงานสรุปเจาะลึก สามารถ Filter ตามประเภทงบ แหล่งงบ และกลุ่มงาน พร้อมระบบ Drill-down ดูรายโครงการ
- Project Tracking & Gantt Chart: ติดตามความก้าวหน้าโครงการ เปรียบเทียบแผนที่วางไว้ (Plan) กับเวลาทำจริง (Actual) พร้อมระบบแจ้งเตือนโครงการล่าช้า

### ✨ 2. ระบบบริหารจัดการแผนปฏิบัติการ (Action Plan / Master Plan Management)

- Smart Form & Cascading Dropdown: ฟอร์มเพิ่ม/แก้ไขโครงการ พร้อมระบบกรองข้อมูลแบบลูกโซ่ป้องกันการเลือกข้อมูลผิดพลาด
- Auto Project ID: ระบบสร้างรหัสโครงการให้อัตโนมัติ (เช่น P-2569-0001) และคำนวณปีงบประมาณอัตโนมัติตามปฏิทิน
- Bulk Update & Auto-Refund: ระบบเลือกหลายรายการเพื่อเปลี่ยนสถานะ หากปรับเป็น "REFUNDED" ระบบจะคำนวณเงินคงเหลือและส่งเข้า "ถังเงินเหลือจ่าย" ให้ทันที
- Batch Import: รองรับการนำเข้าข้อมูลแผนงานทีละมากๆ ผ่านไฟล์ Excel/CSV พร้อมระบบ Validate ข้อมูลก่อนนำเข้าจริง

### ✨ 3. ระบบจัดการธุรกรรมทางการเงิน (Financial Transactions)

- Expense Management: บันทึกการเบิกจ่าย มีระบบ Lock ดักจับไม่ให้เบิกเงินเกินยอดคงเหลือ (ป้องกันงบติดลบ)
- Loan Management: ระบบยืมเงินราชการ รองรับการคืนเงินครบ และคืนบางส่วน พร้อมคำนวณตัดยอดใช้จ่ายจริงให้อัตโนมัติ
- Allocation Top-up: ระบบจัดสรรงบประมาณเพิ่มเติมระหว่างปี เก็บประวัติและเลขที่หนังสืออ้างอิงได้

### ✨ 4. ระบบความปลอดภัยและสถาปัตยกรรม (Security & Database Architecture)

- Role-Based Access Control (RBAC): แบ่งสิทธิ์ผู้ใช้เป็น ADMIN (จัดการผู้ใช้/แผนงาน), EDITOR (ทำธุรกรรม), และ PUBLIC (ดูได้อย่างเดียว)
- Authentication: ยืนยันตัวตนผ่าน Email และมีระบบ Idle Timeout (15 นาที) ตัดเซสชันอัตโนมัติหากไม่มีการใช้งาน
- Dynamic Column Mapping: Backend อ่าน "ชื่อหัวคอลัมน์" แทนการ Fix Index ตำแหน่ง ช่วยให้ผู้ใช้ฝั่ง Google Sheet สามารถแทรก/ย้าย คอลัมน์ได้โดยระบบไม่พัง

### ✨ 5. ประสบการณ์ผู้ใช้งาน (UX/UI Enhancements)

- **Instant Search & Keyboard Navigation:** รองรับการกรองข้อมูลแบบทันทีที่พิมพ์ และสามารถใช้ปุ่ม `Enter` บนคีย์บอร์ดเพื่อสั่งค้นหา หรือบันทึกฟอร์มได้โดยไม่ต้องใช้เมาส์

- **Responsive Design:** หน้าตาเว็บทันสมัย สวยงาม ตัดคำได้พอดี และมี Badge (ป้ายสี) แยกประเภทงบชัดเจน

**สรุปจุดขายหลัก (Selling Point) สำหรับพรีเซนต์:**
"BMD LOEI เป็นระบบที่เข้ามาแก้ปัญหาการทำงานซ้ำซ้อน ลด Human Error ด้วยลอจิกการคำนวณที่แม่นยำ พร้อมทั้งเป็นเครื่องมือชิ้นสำคัญที่ช่วยให้ผู้บริหารเห็นภาพรวมการใช้จ่ายงบประมาณของ สสจ.เลย ได้แบบ Real-time นำไปสู่การตัดสินใจที่รวดเร็วและมีประสิทธิภาพสูงสุด"
